Files:

	- Python Files

		> main.py 				| Main function of program.
		
		> create_table.py 		| Initializes predictive parsing table and saves 
								  into a txt file.
		
		> settingUp.py 			| Initializes text file by removing comments and 						   extra lines.
								  Initializes all variables, inputted reserved words, and input text for parsing.

		> grammar.py 			| Function for checking the grammar of a given 							  input text file.
		
		> errors.py 			| Function for error messages.

		> check_arithmetic.py 	| Function to double check any arithmetic 
								  expressions are valid.
		
		> convertPython.py 		| Convert input text file to Python.

		> outputPython.py       | Inputted text file in Python.

	
	- Txt Files

		> requirements.text		| All required modules for the program.

		> parsing_table.text	| Predictive Parsing Table.

		> math.text				| Arithmetic Expressions from input file.

		> finalp1.text			| Input file with comments and lines.

		> finalp2.text			| Input file without comments and lines.
